# User-Authentication-Portal
User Authentication Portal:

## Description
A secure ASP.NET Web Forms application for user registration and login with session management. Implements password hashing (SHA-256), forgot password functionality, and stores user credentials in SQL Server.

## Features
- User Registration with validation
- Secure Login with password hashing
- Forgot Password functionality
- SQL Server database integration
- Responsive and user-friendly UI with background images

## Tech Stack
- C#, ASP.NET Web Forms
- SQL Server
- HTML, CSS, JavaScript
- ADO.NET
